Stand-alone component
1. Description
In Angular 15, the stand-alone component is out of the developer view. As we mentioned in Notes of Angular v14, it has the following advantages.
  1. For the frictionless component creation
  2. The stand-alone component can simplify learning journey
  3. The stand-alone component can manage dependencies in Component stead of NgModule()
  4. The stand-alone component can be lazy loaded
