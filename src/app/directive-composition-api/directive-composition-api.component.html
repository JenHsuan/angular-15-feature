Directive Composistion API
1. Description
The purpose of the directive composition API is to reuse the directive logic. We no longer have to added the directive on the component to use the directive.
There are a few prerequisites.
  1. The directive has to be a stand-alone directive
  2. The component has to be a stand-alone component
