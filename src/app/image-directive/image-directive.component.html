Description
NgOptimizedImage ensures that the loading of the Largest Contentful Paint (LCP) image is is prioritized.
  1. Automatically setting the fetchpriority attribute on the img tag property (default value of fetchpriority is auto)
  2. Lazy loading non-priority images by default (default value of loading is lazy)
  3. Automatically generating a preconnect link tag in the document head
How to use NgOptimizedImage?
Largest Contentful Paint (LCP) image
The Largest Contentful Paint (LCP) metric reports the render time of the largest image or text block visible within the viewport, relative to when the page first started loading.
Optimized strategies for LCP image
  1. Set fetchPriority to high (NgOptimizedImage: set priority to true)
  2. Set loading to auto (NgOptimizedImage: set loading to auto)
  3. Set the srcset (NgOptimizedImage: property natively according to the width and height)
How to check if the image is FCP
  • Use small image as the cover page (200 * 100): won't be marked as the LCP
  • Use large image as the cover page (2000 * 1000): will be marked as the LCP
