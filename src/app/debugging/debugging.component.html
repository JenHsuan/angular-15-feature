Debug with the Authored Code
1. Description
We are able to navigate to the authored code nstead of the deployed code in the source panel. (Not limited to Angular v15)
2. How to navigate to the authored code from the Chrome DevTool?
  1. Open the Chrome DevTool
  2. Before setting, we could only navigate to the deployed code from the source panel
  3. Enable navigating to the authored code (DevTools > Settings > Experiments > Group sources into Authored and Deployed trees)
  4. After setting, we could navigate to the authored code from the source panel
