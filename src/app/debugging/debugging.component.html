Description
We are able to navigate to the authored code nstead of the deployed code in the source panel. (Not limited to Angular v15)
How to navigate to the authored code
  1. Before setting, we could only navigate to the deployed code from the source panel
  2. Enable navigating to the authored code (DevTools > Settings > Experiments > Group sources into Authored and Deployed trees)
  3. After setting, we could navigate to the authored code from the source panel
